## 🚀 Inicio rápido

### Requisitos previos

- Node.js 18+
- Ionic CLI
- Angular CLI

### Instalación

```bash
git clone https://github.com/flacks-cc/mobile-app.git
cd mobile-app
npm install
ionic serve
```

## 🧩 Tecnologías

| Capa | Tecnología |
