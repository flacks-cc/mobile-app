## Acerca de Flack's Cut & Connect

Aplicación móvil híbrida (PWA) construida con Angular e Ionic, diseñada para clientes de barberías. Permite reservar servicios, explorar productos, gestionar reseñas y administrar la cuenta.

## Funcionalidades

- Explorar servicios y productos
- Realizar reservas
- Dejar reseñas
- Administrar cuenta

## Inicio rápido

### Requisitos previos

- Node.js 18+
- Ionic CLI
- Angular CLI

### Instalación

```bash
git clone https://github.com/flacks-cc/mobile-app.git
cd mobile-app
npm install
ionic serve
```
